#  Smart Expense Analyzer (AI Powered)

A full-stack web application that helps users track expenses, analyze spending patterns, and gain financial insights using AI and machine learning.

## Features
- Add & track expenses
- AI-based automatic expense categorization (NLP + ML)
- Spending insights & analytics dashboard
- Category-wise breakdown (Pie & Bar charts)
- Spending prediction (future estimation)
- Smart financial tips

## AI/ML
- TF-IDF vectorization for text processing
- Naive Bayes model for expense classification
- Pattern-based prediction for future spending
- Anomaly detection (basic)

## Tech Stack
- Frontend: React, Tailwind CSS, Recharts
- Backend: FastAPI
- Database: MongoDB
- ML: Scikit-learn

## How it works
User input → ML model categorizes → Data stored → Insights generated → Charts displayed
